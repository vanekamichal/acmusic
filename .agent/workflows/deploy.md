---
description: Commit and push changes
---

1. Commitni všechny změny do develop branch
2. Pushni změny na remote branch
3. Vytvoř GitHub pull-request z develop do master s krátkým popisem změn
4. Mergni GitHub pull-request do master s pouzitim rebase
5. Aktualizuj (pullni/rebasni) změny z remote master na lokální master branch
6. Smaž lokální develop branch a vytvoř ji znovu z master branch
7. Udělej git fetch pro aktualizaci remote branches
8. Kontroluj kdy doběhne github pages deploy (pres command "gh run watch") a potom mě informuj, že stránky byly nasazeny
9. Na konci vypiš URL stránek do chatu jako odkaz
